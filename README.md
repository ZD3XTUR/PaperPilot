# 🛰️ Research-Pilot Pro
**The High-Performance Intelligence Engine for Modern Researchers**

Research-Pilot Pro is a professional-grade research station designed to unify academic papers, open-source repositories, and technical video content into a single, intelligent interface. It is engineered to optimize the data gathering process and provide rapid intelligence without being hindered by API rate limits.

---

## 🚀 Key Features

| Module | Function |
| :--- | :--- |
| **🌐 Unified Search** | Instant scanning across Google (Web) and YouTube (Video). |
| **🐙 Code Intelligence** | Access to top-tier repositories via GitHub API integration. |
| **📄 Academic Archive** | Real-time paper retrieval from the ArXiv database. |
| **⭐ Personal HUB** | Comprehensive management of Favorites (Stars) and Reading Lists. |
| **⚡ Smart Caching** | Seamless data flow powered by st.cache_data optimization. |

---

## 🛠️ Technical Architecture

The application is built on a Session-Based Caching architecture that minimizes API requests and accelerates user experience.

- Language: Python 3.8+
- Framework: Streamlit
- Protocols: Requests (Session-based), XML ElementTree
- UI Design: Custom CSS Injection (Professional Dark Mode)

---

## 📦 Installation and Setup

To deploy the project in a local environment:

1. Clone the repository:
   git clone https://github.com/ZD3XTUR/PaperPilot.git

2. Install dependencies:
   pip install -r requirements.txt

3. Launch the application:
   streamlit run app.py

## 🎨 Interface Capabilities

- Interactive Sidebar: Manage search history, remove favorites (✖), and mark reading list items as complete (✔).
- Trend Discovery: One-click access to trending sectors such as AI Agents, Quantum Computing, and BioTech.
- Direct Portal: Optimized high-speed redirection cards for YouTube and Web results.
- Connection Reliability: Enhanced error handling for GitHub and ArXiv API nodes.
